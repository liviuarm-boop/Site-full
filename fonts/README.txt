GhidBursa.ro — Self-hosted Fonts
=================================
Download these 5 files and place them in this /fonts/ directory,
then deploy to Cloudflare Pages.

curl -o fonts/plus-jakarta-sans-600.woff2 "https://fonts.gstatic.com/s/plusjakartasans/v8/LDIbaomQNQcsA88c7O9yZ4KMCoOg4IA6-91aHEjcWuA_SG-NSg.woff2"
curl -o fonts/plus-jakarta-sans-700.woff2 "https://fonts.gstatic.com/s/plusjakartasans/v8/LDIbaomQNQcsA88c7O9yZ4KMCoOg4IA6-91aHEjcWuA_Qm-NSg.woff2"

After download, the fonts.css is already configured.
The _headers file already caches /fonts/* for 1 year (immutable).
